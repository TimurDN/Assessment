# Evidence & Reports

This folder holds all UI/UX evidence required by the assessment:

- `screenshots/desktop/` — desktop viewports of each flow step + error/loading/empty states
- `screenshots/mobile/` — mobile viewport equivalents
- `bugs/` — screenshots/videos attached to bug reports in `../bug-reports.md`
- `flow-video.mp4` — 60–120s end-to-end walkthrough
- `lighthouse/` — Lighthouse HTML reports (performance, a11y, best-practices, SEO)
- `accessibility/` — axe-core / Pa11y reports

Populated during the testing phase.
